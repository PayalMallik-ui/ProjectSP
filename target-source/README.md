# Target Source Module

This module contains SailPoint ISC Target Source configurations.

## Overview

Target Sources are the downstream systems where identities and access are managed. These can include WebServices, SaaS applications, and other enterprise systems.

## Contents

### Subdirectories

- **webservice/** - Web service and API-based target integrations
- **saas/** - SaaS application integrations

## WebService Targets

Web service and REST API-based target system configurations.

## SaaS Targets

SaaS application integrations (e.g., Okta, Microsoft Entra, Salesforce, ServiceNow, etc.).
