# SaaS Targets

This subdirectory contains configurations for SaaS application integrations.

## Overview

SaaS targets connect to cloud-based applications and services for identity and access management.

## Supported Platforms

- Okta
- Microsoft Entra ID (Azure AD)
- Salesforce
- ServiceNow
- Workday
- Jira Cloud
- And other SaaS applications

## Contents

- SaaS connector configurations
- Application-specific settings
- Authentication and API credentials
- Account schema definitions
- Provisioning and deprovisioning rules
- Role mapping configurations
