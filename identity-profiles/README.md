# Identity Profiles Module

This module contains SailPoint ISC Identity Profile configurations.

## Overview

Identity Profiles define templates for different types of identities in your organization (e.g., employees, contractors, customers). They determine how identities are created and managed within ISC.

## Contents

- Identity profile definitions
- Attribute mappings
- Lifecycle rules
- Access assignment policies
