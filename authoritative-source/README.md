# Authoritative Source Module

This module contains SailPoint ISC Authoritative Source configurations.

## Overview

The Authoritative Source is the primary system of record for identity data. This is typically your HR system or master identity database that serves as the source of truth for identity information.

## Contents

- Authoritative source connector configurations
- Attribute mappings from source system
- Account creation and lifecycle rules
- Identity refresh schedules
- Correlation rules
