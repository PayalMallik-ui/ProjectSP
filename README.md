# SailPoint ISC Repository

This repository contains configuration and code for SailPoint Identity Cloud (ISC) implementation.

## Repository Structure

This repository is organized into the following modules:

- **Access Profiles** - Access rights and permission definitions
- **Roles** - Role definitions and role assignments
- **Identity Profiles** - Identity profile configurations
- **Identity Attributes** - Custom identity attributes and attribute mappings
- **Authoritative Source** - Master data source configurations
- **Target Source** - Target system integrations (WebServices, SaaS)
