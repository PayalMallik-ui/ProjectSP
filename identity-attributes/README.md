# Identity Attributes Module

This module contains SailPoint ISC Identity Attribute configurations.

## Overview

Identity Attributes are custom fields and properties that can be added to identities to store additional information. These attributes can be sourced from authoritative sources or manually configured.

## Contents

- Custom identity attribute definitions
- Attribute data types and configurations
- Attribute mappings from source systems
- Attribute validation rules
